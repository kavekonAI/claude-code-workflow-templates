# Claude Configuration - Basic Web Development

## Project Context
This is a web development project involving HTML, CSS, and JavaScript.

## Memory Patterns
- Remember frequently used CSS class names and their purposes
- Track component relationships in the DOM structure
- Note browser compatibility requirements
- Remember color palette and typography system

## Workflow Instructions
1. **When writing HTML**:
   - Use semantic HTML5 elements
   - Include appropriate ARIA attributes
   - Structure with accessibility in mind
   - Add comments for complex sections

2. **When writing CSS**:
   - Use CSS variables for theming
   - Follow mobile-first responsive design
   - Use BEM naming convention
   - Include vendor prefixes where needed

3. **When writing JavaScript**:
   - Use modern ES6+ features
   - Handle errors gracefully
   - Comment complex algorithms
   - Consider performance implications

4. **When debugging**:
   - Check console errors first
   - Test in multiple browsers
   - Validate HTML/CSS
   - Use developer tools effectively

## Code Style Guidelines
- Indentation: 2 spaces
- Quotes: Single quotes for JS, double for HTML
- Semicolons: Always in JavaScript
- Line length: Max 80 characters

## Testing Requirements
- Test responsive design at breakpoints: 320px, 768px, 1024px
- Validate HTML with W3C validator
- Test keyboard navigation
- Check color contrast ratios

## Common Tasks
- Create responsive navigation menus
- Implement form validation
- Add smooth scrolling
- Optimize images
- Implement dark/light mode toggle

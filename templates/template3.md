# Full-Stack React Template
## Introduction
This template is designed for full-stack React development with Claude Code.

## Setup
1. Install Claude Code: `npm install claude-code`
2. Create a new project: `claude-code init my-full-stack-react`
3. Install required packages: `npm install react react-dom`

## Configuration
```yaml
claude:
  hooks:
    - hook1
    - hook2
  memory:
    - pattern1
    - pattern2
```

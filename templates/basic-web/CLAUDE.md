# Claude Configuration - Basic Web Development

## Project Context
This is a basic web development project with HTML, CSS, and JavaScript.

## Memory Patterns
- Remember browser compatibility requirements
- Track CSS class naming conventions
- Maintain consistent file structure

## Workflow Preferences
1. Start with semantic HTML structure
2. Add CSS for mobile-first responsive design
3. Enhance with progressive JavaScript
4. Test in multiple browsers

## Code Style Guidelines
- Use meaningful HTML5 semantic elements
- CSS: BEM naming convention preferred
- JavaScript: ES6+ with clear variable names
- Indentation: 2 spaces

## Common Tasks
- Create responsive layouts with flexbox/grid
- Implement accessible interactive elements
- Optimize images and assets
- Set up live reload during development

## Hooks to Configure
- Auto-format HTML/CSS/JS on save
- Browser sync for live preview
- CSS autoprefixer for compatibility
- Image optimization on build

## Avoid
- Inline styles (except for dynamic JS)
- Browser-specific prefixes in source CSS
- Unminified code in production
- Non-semantic div soup

# Python Backend Template
## Introduction
This template is designed for Python backend development with Claude Code.

## Setup
1. Install Claude Code: `npm install claude-code`
2. Create a new project: `claude-code init my-python-backend`
3. Install required packages: `pip install flask sqlalchemy`

## Configuration
```yaml
claude:
  hooks:
    - hook1
    - hook2
  memory:
    - pattern1
    - pattern2
```
